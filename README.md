# VK Турагентство Бот — v3

## Что изменилось в этой версии

### 1. Выбор страны кнопками
Вместо текстового ввода пользователь выбирает направление из готовых кнопок:

```
🇹🇷 Турция    🇪🇬 Египет
🇹🇭 Таиланд   🇦🇪 ОАЭ
🇬🇷 Греция    🇨🇾 Кипр
🇲🇻 Мальдивы  🇪🇸 Испания
🔙 Главное меню
```

Чтобы добавить новую страну — достаточно дописать строку в список `DESTINATIONS` в `bot.py`:
```python
DESTINATIONS = [
    ("🇹🇷", "Турция"),
    ("🇨🇺", "Куба"),   # ← добавить сюда
    ...
]
```

### 2. SQLite база данных (`bot_data.db`)

База создаётся автоматически при первом запуске рядом с `bot.py`.

**Таблицы:**

| Таблица | Что хранит |
|---|---|
| `user_states` | Текущий шаг диалога каждого пользователя (JSON) |
| `users` | Имя пользователя, дата первого и последнего визита |
| `search_history` | Параметры каждого поиска тура + количество результатов |

**Состояния диалога** теперь переживают перезапуск бота — пользователь продолжит с того же шага.

**Имя пользователя** кэшируется в `users`, чтобы не запрашивать VK API при каждом сообщении.

**История поисков** позволяет видеть, какие направления и параметры популярны.

Посмотреть данные можно любым SQLite-клиентом (например, DB Browser for SQLite) или через консоль:
```bash
sqlite3 bot_data.db "SELECT * FROM search_history ORDER BY searched_at DESC LIMIT 20;"
sqlite3 bot_data.db "SELECT * FROM users;"
```

## Структура проекта

```
vk_tour_bot/
├── bot.py            # Основной файл (роутинг, клавиатуры, отправка)
├── tour_selector.py  # База туров + логика подбора (здесь photo_id)
├── doc_consultant.py # Ответы по документам и визам
├── user_state.py     # SQLite: состояния, пользователи, история поисков
├── config.py         # Токен VK и ID администратора
├── requirements.txt  # Зависимости
├── bot_data.db       # База данных (создаётся автоматически)
└── README.md         # Эта инструкция
```

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

## Как добавить фотографии к турам

1. Загрузите фото отелей в альбом группы ВКонтакте
2. Откройте фото — скопируйте из URL часть `photo-XXXXXXXXX_XXXXXXXXX`
3. Вставьте в `tour_selector.py` в поле `"photo_id"` нужного тура
